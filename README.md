# Project on vehicle identification and counting using YOLO

📌 [Link drive tải files](https://drive.google.com/drive/folders/1wTHMOwCS9XXA8ZfkDMAu-3acf1Lk7SMM?usp=sharing)

📌 Cài đặt các thư viện:
- python -m venv venv ➡ tạo môi trường ảo
- pip install -U ultralytics ➡ thư viện chạy YOLOv8
- pip install pillow ➡ chạy cho file UI **gui_app.py** (tkinter thường đã có sẵn khi cài Python)
